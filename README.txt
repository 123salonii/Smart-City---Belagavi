Belagavi Smart City is a web-based guide showcasing Belagavi's tourist attractions, local food, shops, and emergency services.
It includes dedicated pages for events, places to stay, transportation options, and scenic spots like waterfalls.
The project features a gallery of images and a CSS stylesheet for a responsive and visually engaging experience.
Designed as a tourism-focused platform, it aims to enhance digital accessibility and promote Belagavi as a smart city destination
